### 소개글
--- 

인프런 강의로 TMDB API을 활용한 영화웹 프로젝트를 만드는 중 백엔드 개발에서 프론트까지 시장 가치을 높이기 위해 NextJS로 나만의 프로젝트를 시작하게 되었습니다.

TMDB API를 활용해 영화별 예산, 수익, 수익률 차트를 넣어 시각적으로 표현하고 영화의 다른 정보(영화개봉일, 출연진, 트레일러, 좋아요)를 보여주는 사이트입니다.
